# metadata-scrubber
Command-line tool for batch removal of metadata from image files. Supports JPEG, PNG, TIFF, WebP. Zero dependencies beyond Python standard library.

## Usage
python scrub.py --input ./photos/ --output ./clean/ --verify

## Features
- Strips all EXIF, IPTC, XMP metadata
- Verification mode confirms removal
- Batch processing with progress bar
- No external dependencies

Built for personal use. No telemetry. No logging. MIT license.

# dns-leak-checker
Simple script to verify DNS queries are routed through expected resolver. Designed for post-connection verification after VPN or Tor setup.

## Usage
python check.py --expected-resolver 10.64.0.1

Outputs pass or fail. No data collection. No network calls beyond DNS resolution test.

Differential privacy sounds promising in theory. Implementation is problematic. Epsilon values in production are often too generous. Real protection requires strict bounds that reduce utility significantly.

Certificate transparency is a double-edged sword. It prevents rogue certificates. It also exposes every subdomain you register. For organizations requiring operational security this is a problem.

Zero-knowledge proofs have applications beyond cryptocurrency. Authentication without credential exposure. Age verification without identity disclosure. The technology exists. Adoption is the challenge.

Metadata analysis is underestimated as identification vector. Posting times alone can narrow timezone. Combined with stylometric features the narrowing becomes significant. Content encryption solves nothing if metadata is exposed.

Supply chain attacks on open-source dependencies are increasing. The xz utils incident demonstrated the vulnerability clearly. Code review alone is insufficient. Reproducible builds help but adoption remains limited.

End-to-end encryption in Matrix works well. The Megolm protocol handles group encryption efficiently. Key verification via emoji comparison is user-friendly. Most users skip it. This is a problem.

Self-hosting a Matrix server is recommended for serious privacy needs. Synapse is resource-heavy. Dendrite is improving rapidly. Worth the setup effort for those with technical capability.

Federated systems have trade-offs. Decentralization improves resilience. It also complicates moderation. Each server operator sets their own policies. Consistency is not guaranteed.
