# Beam Calculator v0.2
Simple Python tool for calculating bending moments and shear forces in simply supported beams. Built for civil engineers who want to automate repetitive structural calculations.

## Features
- Point loads and distributed loads
- Support reactions calculation
- Bending moment diagrams (matplotlib)
- Export results to CSV

Still learning Python so the code is probably not great, but it works for my use cases. Feedback welcome.

I'm trying to calculate the deflection of a cantilever beam with a point load using Python. My formula seems correct (PL^3/3EI) but the output is off by a factor of 1000. I suspect it's a unit conversion issue - I'm mixing mm and m somewhere. Can someone spot the error?

Update: Found it. I was inputting the moment of inertia in mm^4 but the length in meters. Converting everything to consistent units fixed it. Leaving this here in case someone else makes the same mistake.
