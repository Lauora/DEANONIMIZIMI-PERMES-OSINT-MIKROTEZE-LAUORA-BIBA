# OSINT-Research
Ky punim propozon dhe validon një framework gjashtë-vektorial OSINT për de-anonimizimin e identiteteve online, duke demonstruar nëpërmjet gjashtë profileve sintetike se si niveli i mbrojtjes dixhitale ndikon në mundësinë e identifikimit, me rezultate që variojnë nga 84/100 (zero mbrojtje) deri në 5/100 (mbrojtje maksimale).
